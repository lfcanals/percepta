Runtime objects to get realtime information from connected camera


# Implemented features
1. Calibration of the camera
2. Calibration of the road borders 
3. Realtime lane detection 
4. Print of the border lanes to verify borders - just for testing


# Next features
5. Different color for lane close to the border and over the border
6. Sound through left / right speaker when we get close to the border
